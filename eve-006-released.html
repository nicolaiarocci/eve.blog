Eve 0.0.6 Released

Lun 13 Maggio 2013 By Nicola Iarocci

The two main features introduced by this release are Event Hooks and Projections. Event hooks allow to subscribe to GET, POST, PATCH, DELETE requests with mutiple callbacks. Projections are conditional queries where the client dictates which fields should be returned by the API (when projections are omitted the whole document is returned).

We also added support for HEAD requests, made sure that only fields defined in resource schemas are returned and applied a bunch of performance optimizations, especially with regard to POST (Eve now performs bulk database inserts) and ETags computation. Also, if you have been experiencing 400s make sure to download this release, as a couple of significant bugs have been squashed. See the changelog for all the news.
